# **App Name**: GameParadise

## Core Concepts:

- **Game Lobby**: A central, welcoming hub where players can easily choose from a variety of mini-games. The lobby will feature large, clickable cards for each game, showcasing a fun icon and a brief description.
- **Variety of Games**: The platform will host multiple simple and engaging games, starting with "Space Shooter" and "Bubble Burst." New games can be added in the future.
- **Child-Friendly Interface**: The entire user interface will be designed with children in mind. This means bright colors, large and easy-to-read text, and intuitive navigation.
- **Scoring and Progression**: Each game will have its own scoring system to encourage replayability. High scores could be saved locally to track personal bests.
- **Responsive and Accessible Design**: The application will be fully responsive, providing a seamless experience on desktops, tablets, and smartphones.
